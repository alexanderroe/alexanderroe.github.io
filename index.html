Major/Minor Progress Checker.
A web application that creates a data visualization, which shows
you which minors at UIUC you are closest to completing, based on
the classes you've already taken. The source code is on GitHub.

Command Line Image Viewer (cliv).
A small utility program I call 'cliv', which displays images directly in the terminal,
albeit at a reduced resolution. Useful for quick previews of image files without manually
opening the image in an external application. The source code is on GitHub.

Notes on Ordinary Differential Equations.
Based on Math 441 at UIUC, these LaTeX notes cover topics like
ODEs, systems of ODEs, and linearization.
